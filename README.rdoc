This application is build on jRuby on Rails.

Rails-4.2.0 version and jRuby -1.7.24 which is equivalent to Ruby 193.

This SPA is a search engine which crawls data from different websites and shows in the search results.

It uses AngularJS,jQuery ,HTML/CSS at the FE and Solr at the backend.

The application is done at a very basic level, for just testing the crawling .
